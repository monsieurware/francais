Diagnostics

Every packet is logged below. On connect the app reads the bed's characteristics to enable control, then sends commands in the selected protocol. Default is DewertOkin, the family your CB3332 belongs to.

If writes show as TX but the bed does not move, switch the pipe and re-test a preset. Default is Nordic UART because the 5A...A5 framing looks like a serial protocol.

Raw hex sender. Space or comma separated. A bare action byte like 00 is auto-wrapped in the 5A...A5 frame.

Turn massage on, then tap Find it. It steps through likely off codes with a pause between each. Watch the bed, note which one stops the motors, then type that value above and it will be used by the Off button and the timer.

Reverse-engineered, unofficial. Do not run motor controls with someone in the bed during testing.
